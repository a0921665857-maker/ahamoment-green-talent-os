# 文章 1:〈為什麼看似弱勢的人,反而更可能談成?拆解真正的談判籌碼〉

## 版本 A｜反共識 / 犀利｜建議 18:00–19:00 或 21:00

在 INSEAD 學談判,最反直覺的一課:

你的籌碼,不在你手上。
在「對方失去你之後要賠多少」裡。

大多數人一遇到自己理虧、資源少,就直接進入求饒模式。
但求饒 = 把控制權整碗交出去。

真正該問的不是「我有沒有籌碼」,是:
**如果這件事破局,對方損失多少?**

一個比喻我一直記得——
一隻會下蛋的母雞,跟一鍋煮好的雞湯。
湯有立即的價值;但只要雞蛋的總產值大於那鍋湯,沒人會想殺母雞。

把你自己代入那隻母雞。
你「活著、繼續把事做好」能創造的價值,減掉「換掉你、重來」的成本——只要還是正的,對方就有結構性的理由幫你,而不是丟掉你。

所以底氣從來不是「我多強」。
是「我已經把你的損失算清楚了,我知道你不想走那條路」。

你現在生活或工作裡,有沒有一個你一直覺得「自己沒籌碼」的局?
也許,你只是還沒算清楚對方的損失而已。

# 文章 3:〈為什麼你越努力,反而越不賺錢?拆解固定成本謬誤〉

為什麼你越努力,反而越不賺錢?

因為你把「有收入」當成「有賺錢」。

錢進來了 ≠ 你賺了。賺錢是:進來的錢,大於你為它付出的所有成本——包括沒人幫你列在發票上的隱形成本(改稿到三點、客戶每週 Line 你七次、壓力大到週末整個廢掉)。

把你的客戶從最賺錢排到最賠錢(不是看誰付最多,是扣掉真實成本後剩多少)。

你會發現:20–30% 的客戶撐起你大部分獲利;末端那幾個,每服務一次,你的淨值就悄悄少一點。

砍掉那個「毒瘤客戶」,是你今年能給自己最快的加薪。

而且「固定成本」沒你想的固定——拉長到 3–6 個月,租約、人力、外包幾乎都能調。真正卡死你的,是你把它當成不能動的前提,然後一筆筆接下「分攤成本用」的爛訂單。

學會在對的時機說不,比學會把握每個機會,難太多了。

# 文章 4:〈別再執著 100%:縮小分母,才是真正的財富邏輯〉

大多數人追「賺更多的錢」。
真正厲害的人追「讓每一塊自己出的錢,產生更大的倍數」。

聽起來一樣,其實一個追總量,一個追效率。

報酬率 = 報酬 ÷ 投入。多數人拚命做大分子(賺更多),但分子有極限。

真正的魔術,是把分母做小。

同一個專案、同樣獲利:你出 1000 萬,報酬率 X;你用設計只出 100 萬拿到類似的份,報酬率就是 10X。

怎麼把分母做小?用「出力」取代「出錢」。
你在某個領域有別人沒有的能力或品牌,那就是一張零成本的籌碼——可以換到位置、換到利潤分配,不用砸大錢就能參與。

所以問題不是「我有沒有足夠的錢」,是「我有沒有別人願意出錢來換的東西」。

這兩個問題,方向完全相反。

**↳ 留言**:你最大的零成本籌碼,是你的專業——但前提是市場看得懂它值多少。先看清你的定位 👉 連結

# 文章 12:〈當公司快倒閉,誰才是真正的老闆?〉

你手上的股票,某種程度上只是一張「排隊號碼牌」。

公司賺錢時你是股東;一旦資產不夠還債,股東的請求權會自動失效——這是你買股票時,合約裡就寫好的規則,只是沒人特別告訴你。

把公司想成一棟會漏水的公寓:一樓銀行、二樓無擔保債主、三樓債券、頂樓股東。水位從頂樓往下淹。

真正有籌碼的,是「剛好被淹到一半」的那一層——損失慘重到足以主張重新分配,又還沒全滅到什麼都不剩。那群人才是談判主角,而且通常不在公開股東名單上。

所以在職場看「誰說了算」,別只看誰頭銜最大、股份最多。要問:危機發生時,誰手上有那份能掀桌子的文件?

那個人,才是真正的玩家。

# 文章 13:〈老闆桌下的隱形狗鍊:你以為你簽的是貸款,其實你簽的是控制權〉

你以為「擁有」代表「控制」。在商業世界,這兩件事可以完全分開。

企業貸款不是「拿錢 → 還錢」那麼單純,它是一份「持續表現合約」:負債不能超過獲利幾倍、現金流不能低於某水位、核心資產不能偷賣。

踩到那條線,叫「技術性違約」——你現金很夠、利息照繳,但負債比例破表,你就違約了。什麼都沒發生,但你違約了。

然後債權人有權介入:要求說明、換管理層、凍結決定。

想像一台自駕車:平常你握方向盤,但系統一直在背景監測;一旦某個數字超標,方向盤自動被接管,不需要你同意、不需要開會表決。

所以下次看到某個 CEO 在台上講得頭頭是道,別急著相信他還握著方向盤。

**↳ 留言**:看清「誰真的握著方向盤」是一種生存能力。用回你自己的職涯:你現在的「決定空間」有多大、被市場定位成什麼?先看清 👉 連結

# 文章 14:〈債務壓抑:有一種努力,叫做在幫債主打工〉

有一種人,突然就開始混、消極、對什麼都興致缺缺。你以為他變了、燒盡了。

但他可能只是做了一個很理性的計算。

決定一個人要不要努力的,從來不是「有沒有機會」,是「機會做成了,好處歸誰」。

INSEAD 教授問過一個讓全教室安靜的問題:
「如果你今年所有年終,依法都會被拿去還你過去欠的債,你今年還會拚命工作嗎?」

那五秒的沉默,比任何課本都誠實。

職場版:你接手一個爛攤子,前任的 KPI 缺口大到你補三年都不夠,救活了功勞還算「填坑」、不算你的成就——你會全力以赴,還是默默更新 LinkedIn?

你的「沒動力」,很多時候不是懶,是誘因缺失的正常反應。

捫心自問:你現在的努力,有多少比例的果實,確定是你的?
