# 文章 1:〈為什麼看似弱勢的人,反而更可能談成?拆解真正的談判籌碼〉

## 版本 A｜反共識 / 犀利｜建議 18:00–19:00 或 21:00

在 INSEAD 學談判,最反直覺的一課:

你的籌碼,不在你手上。
在「對方失去你之後要賠多少」裡。

大多數人一遇到自己理虧、資源少,就直接進入求饒模式。
但求饒 = 把控制權整碗交出去。

真正該問的不是「我有沒有籌碼」,是:
**如果這件事破局,對方損失多少?**

一個比喻我一直記得——
一隻會下蛋的母雞,跟一鍋煮好的雞湯。
湯有立即的價值;但只要雞蛋的總產值大於那鍋湯,沒人會想殺母雞。

把你自己代入那隻母雞。
你「活著、繼續把事做好」能創造的價值,減掉「換掉你、重來」的成本——只要還是正的,對方就有結構性的理由幫你,而不是丟掉你。

所以底氣從來不是「我多強」。
是「我已經把你的損失算清楚了,我知道你不想走那條路」。

你現在生活或工作裡,有沒有一個你一直覺得「自己沒籌碼」的局?
也許,你只是還沒算清楚對方的損失而已。

## 版本 B｜故事 / 走心｜建議 21:00–22:00

在 Deloitte 那年,我幫一個客戶做碳盤查,做到一半他們財務出問題、預算被砍,計畫差點喊停。

我的第一直覺是:道歉,然後趕快配合調整。

現在回想,那是最弱的一步。

如果我當時早點想清楚一件事——
「換掉我們團隊的交接成本是什麼?已經建好的資料架構,外部顧問接手有多難?」
我說話的方式會完全不一樣。

那不是傲慢。那只是把一件本來就是事實的事,說出來而已。

我在 INSEAD 談判課學到的核心是:
**你的籌碼,不是你擁有多少,而是對方失去你要賠多少。**

會求饒的人,往往是還沒幫對方算那筆賬的人。

下次你覺得自己「沒底氣」開口時,先別道歉。
先問:如果我現在消失,對方的麻煩是什麼?

## 版本 C｜清單 / 乾貨｜建議 22:30–23:30

90% 的人談判會輸,不是不夠強,是卡在這 3 個思考陷阱(INSEAD 談判課拆解):

**1. 以為弱勢就沒籌碼。**
就算你口袋是空的,只要對方失去你的代價夠高,你就有談的空間。籌碼藏在「對方還沒算清的那筆損失」裡。

**2. 太早攤牌、太晚準備。**
有句話我一直記得:「先關上所有緊急出口,再大喊失火。」先備好解決方案,再去開那個你一直在拖的困難對話。

**3. 以為談判是要贏過對方。**
不是。目的是讓對方覺得「跟你合作」比「不合作」划算。讓對方覺得自己輸了,他下次不會再跟你合作。

**10 分鐘練習:**
① 寫下一個對你不利的局。
② 別想你要說什麼,先想:如果現在破局,對方損失多少?
③ 找一件「只有你能做」、能幫對方減少損失的事。
④ 用一句話把它包成邀請,不是道歉:「我想提一個對兩邊都有利的方案。」

做完你可能會愣一下。那個愣住的感覺,就是籌碼。
