# 爆文模板庫（Viral Template Bank）

這是發行機器的心臟。每天的熱榜掃描往這裡餵新樣式，每週日的成效閉環用 Michael 自己的數據幫模板記分。模板夠多、輪替夠亂，讀者才不會發現這是一台機器。

## 使用規則（引擎每天照做）

**輪替鐵律：**
1. 同一模板 10 天內不重用（查底部「使用紀錄」）
2. 同一大類不連續兩天使用
3. 標「❄️ 冷凍」的模板跳過，直到有新證據解凍

**表面特徵擲骰（每篇隨機組合，這是反模板偵測的關鍵）：**
- 字數帶：短（150 字內）／中（300 字上下）／長（450 到 500 字），三天內不重複同一帶
- 形式：單則／2 到 3 則串接，串接斷點照 playbook（假動作前、轉折前）
- 斷行密度：密（幾乎一行一句）／疏（自然段落），交錯用
- 結尾手勢從備選庫輪替，同一句 14 天內不重用：轉發給那個朋友／收藏起來／同意就轉／你呢，留言告訴我／先存著，下次投履歷前看一眼／傳給那個一直說想離職的同事
- emoji：預設零。每 4 到 5 篇允許一篇用一個（放在數字或榜單旁），絕不放句尾

**內容鐵律（不擲骰，永遠成立）：**
全形標點；零破折號「——」零「----」；無雞湯開頭、無三句以上排比、無凱旋結尾；「不是⋯是⋯」全文限一次且只在落點；個人故事只取 material-bank ✅ 條目；連結只放首則留言。

## 模板總表（37 個起跳，每日掃描持續新增）

每個模板的欄位：鉤子（第一行長什麼樣）／脊椎（往下怎麼走）／語氣／素材接口／時段。
「證據」欄：假設 = 來自研究未實測；實測 = 被 threads-log 數據支持過。

### 一、故事敘事類（身分工作，最強帶入感）

**T01 對白冷開場**
鉤子：第一行是一句別人說的話，帶引號，不解釋誰說的。
脊椎：對白 → 我愣住的內心一句 → 回頭鋪場景 → 轉折 → 安靜身分句。
語氣：低溫回憶。素材：material-bank A2、A4、面試與客戶對話。時段：21:00 到 22:30。證據：假設（playbook 列為最強滑停）。

**T02 時間地點身體感**
鉤子：時間＋地點＋一個身體動作（凌晨兩點，辦公室只剩我桌上那盞燈）。
脊椎：場景 → 沈默念頭 → 假動作 → 轉折 → 落點。標準五拍。
語氣：沈浸。素材：material-bank A1、A3。時段：21:00 到 22:30。證據：假設。

**T03 物件錨點**
鉤子：一個小物件承載全篇（那杯涼掉的咖啡、印到一半的報告、沒寄出的辭職信）。
脊椎：物件 → 物件背後那一刻 → 事件 → 物件在結尾再出現一次收束。
語氣：電影感。素材：任何 ✅ 故事的道具細節。時段：21:00 到 22:30。證據：假設。

**T04 朋友求助**
鉤子：朋友傳來的一句訊息（「我又沒上。」）。
脊椎：訊息 → 我知道他很強 → 找到他輸在哪 → 那個原因跟能力無關 → 鏡子句遞給讀者。
語氣：替人不平。素材：material-bank B 類（必須匿名化到認不出）。時段：21:00 到 22:30。證據：假設。

**T05 結局倒敘**
鉤子：先講結果的一行（我拒絕了那個年薪多五成的 offer）。
脊椎：結果 → 讀者以為的原因 → 真正的原因 → 過程 → 落點呼應開頭。
語氣：懸念。素材：✅ 轉職決策點。時段：21:00 到 22:30。證據：假設。

**T06 低谷自白**
鉤子：被拒絕或搞砸的那一刻，原話重現（「我們可以直接結束了。」）。
脊椎：低谷 → 那晚的自我懷疑 → 不急著反轉，先待在谷底兩行 → 小小的往前一步 → 未完成式結尾（不凱旋）。
語氣：赤裸但克制。素材：material-bank A4（⚠️ 待 Michael 確認才可用）。時段：21:30 到 22:30。證據：假設（A4 原始版本是他發過表現最好的）。

**T07 旁觀切片**
鉤子：我在某個公共場合看到的一幕（捷運上、咖啡店、機場登機口）。
脊椎：那一幕 → 讓我想起某件事 → 觀點 → 把鏡子遞給讀者。
語氣：觀察者。素材：真實見聞（沒把握就標 [待補真實細節：__]）。時段：任意。證據：假設。

### 二、數字衝擊類（乾貨＋反差）

**T08 孤數開場**
鉤子：第一行只有一個數字加句號（544%。）。第二行才解釋。
脊椎：數字 → 它是什麼 → 大家的直覺解讀 → 數字背後反直覺的那一面 → 一句行動。
語氣：冷靜丟證據。素材：薪資報告（台灣綠領機會 +544% 溢價僅 +5.3%）、生活成本（名目 2.4 倍實拿 1.6 倍）。時段：18:00 到 19:00 或 22:30。證據：假設。

**T09 兩地對比帳**

脊椎：名目差 → 一項一項扣（房租、稅、生活）→ 實際差 → 所以真正該問的問題。
語氣：幫你算帳的朋友。素材：cost-of-living 實查數據、薪資報告星台對比。時段：18:00 到 19:00。證據：假設。

**T10 成本拆解**
鉤子：你以為賺到 X（一行），其實剩 Y（一行）。
脊椎：表面數字 → 隱形成本逐項現形 → 剩多少 → 這跟你的哪個決定有關。
語氣：拆穿但不酸。素材：MBA ROI 計算器邏輯、固定成本謬誤（blog 文 3）。時段：18:00 到 19:00。證據：假設。

**T11 時間帳本**
鉤子：幾個年份直排，每年一行一件事（2019 四大。2021 想逃。2023 INSEAD。）。
脊椎：年表 → 停在其中一年放大那一刻 → 當年不知道的事 → 現在才看懂的線。
語氣：回望。素材：material-bank A5 完整弧線。時段：21:00 到 22:30。證據：假設。

**T12 榜單體**
鉤子：一個排行榜的標題行（綠領職位薪資，我看過的前五名）。
脊椎：5 到 3 名快速過 → 第 2 名稍停 → 第 1 名意外＋解釋為什麼 → 一句觀點。
語氣：行內人揭牌。素材：greenJobs 點評、薪資報告分位數。時段：22:30 到 23:30。證據：假設。

**T13 對照表**
鉤子：兩欄對照的第一組（大家找工作看：薪水。行內人看：誰付錢給這個部門。）。
脊椎：三到四組對照，每組錯位遞增 → 最後一組最痛 → 收一句。對照組內容每組句式要不同，避免排比。
語氣：內行 vs 外行。素材：JD 翻譯器洞察、job-pick-playbook 六動作。時段：22:30 到 23:30。證據：假設。

### 三、反共識觀點類

**T14 你以為其實**
鉤子：一個普遍信念直述（大家都說先考證照再轉行）。
脊椎：信念 → 我也信過＋付出的代價 → 翻面的證據 → 新框架一句話。「不是⋯是⋯」只在此處用一次。
語氣：過來人。素材：證照迷思（A3）、blog 文 10、11。時段：18:00 到 19:00。證據：假設。

**T15 集體錯誤**
鉤子：一個大家偷偷都在做的行為，具體到有畫面（把「主導」寫成「參與」）。
脊椎：行為 → 為什麼我們都這樣（文化根源一行）→ 代價 → 怎麼改，給一個可立刻做的動作。
語氣：說中你。素材：謙虛稅（B6）、履歷誤讀。時段：18:00 到 19:00。證據：假設。

**T16 打臉過去的自己**
鉤子：我以前也覺得 X（一行），直到付出代價。
脊椎：舊信念 → 撞牆那一刻（具體場景）→ 修正後的信念 → 不勸你，只說我的帳。
語氣：自首。素材：任何 ✅ 轉折點。時段：18:00 到 19:00 或 21:00。證據：假設。

**T17 行內真話**
鉤子：這行有件事，圈內人都知道但沒人在外面說。
脊椎：懸念一行 → 真話 → 為什麼沒人說 → 知道這件事你該怎麼用。
語氣：壓低聲音。素材：碳市場、ESG 顧問業、MBA 申請的實況（必須真實，不可誇大）。時段：18:00 到 19:00。證據：假設。

**T18 潑冷水**
鉤子：先別急著＋一個大家正衝的動作（先別急著裸辭去念 MBA）。
脊椎：熱門動作 → 我看過的翻車帳（匿名化）→ 什麼情況才值得 → 一個判斷標準。
語氣：擋在路口的朋友。素材：MBA ROI、EP 四年門檻、生活成本。時段：18:00 到 19:00。證據：假設。

**T19 重新定義**
鉤子：一個常用詞＋「不是你以為的意思」的變體寫法。
脊椎：詞 → 大家的用法 → 行內的真實定義 → 定義換了之後你的決定會怎麼變。
語氣：換眼鏡。素材：blog 文 1（籌碼）、文 6（賣掉）、文 12（老闆）。時段：18:00 到 19:00。證據：假設。

**初始假設（來自 playbook 與研究，等 threads-log 實測數據逐條驗證或推翻）：**
- 對白開場是最強滑停，時間地點身體感次之
- 故事 > 清單 > 驚悚（對「強但被誤讀」的受眾）
- 安靜身分句結尾 > 凱旋結尾
- 斷則斷在假動作前、轉折前

**外部實測支持（2026-07-16 盲點研究，證據等級高於假設、低於自家數據）：**
- 台灣互動峰：週四 20:00 最高，週三四 20:00 到 21:00 次之；週末低谷，例外是週日 21:00 收心峰（QSearch，3,600 萬則台灣貼文）
- 互動權重：私訊分享／轉發 > 深度留言（5 字以上，約 30 個讚）> 一般留言 > 讚；快速滑過是負信號
- 發文後 30 到 90 分鐘的互動速度決定推不推陌生人；有回覆經營的貼文 24 小時互動高 2.3 倍
- 回覆別人的總價值約等於發文（Mosseri）；在熱帖留言導流可優於自行發文（台灣 120→10,000 粉案例）
- 連結降權已於 2025 官方修正，帶連結貼文互動 +17%（連結位置 A/B 實驗中，見 thread-pusher）
- 同一 topic tag 連用一週有被隱藏一個月的實測案例；一篇限一個 tag
- 超過 500 字被系統自動切則（硬上限）
