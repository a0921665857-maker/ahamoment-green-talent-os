# 真實素材庫 v2（Material Bank）

引擎規則：
1. `/thread` 與 thread-pusher 產文時，**第一人稱親身故事只能用本檔標 ✅ 的條目**。
2. Michael 確認方式：把 ⬜ 改 ✅、直接改寫細節、或整條刪掉。隨時可加新條目。
3. **訪談庫（D 區）**是本檔的進料口：Michael 在對話裡回答題目，引擎整理成新條目入庫（預設 ✅，因為是他親口說的）。
4. **人格側寫（E 區）**由引擎維護：每次訪談、每篇定稿、每次 Michael 改稿都往裡蒸餾，目標是能模擬他的判斷與語氣。
5. **模擬素材（F 區）**：引擎基於側寫產的「還沒發生但會發生」素材。使用鐵律見該區。

分類：
- **A 類 = 親身經歷**（第一人稱發）
- **B 類 = 他人故事**（發文必須匿名化到當事人認不出）
- **C 類 = 觀點/框架**（出自他的 blog，天然已確認）
- **D 類 = 訪談答案**（他親口說的，整理後編入 A/B/C 或獨立成條）
- **E 類 = 人格側寫**（不直接發文，供模擬用）
- **F 類 = 模擬素材**（引擎推演，有使用限制）
