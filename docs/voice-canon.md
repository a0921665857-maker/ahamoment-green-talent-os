## 正典一（2026-07-13 發）：星台薪資對比 — **目前最強**

**成效（API 實拉 2026-07-17）：9,556 views｜188 讚｜9 回覆｜1 轉發｜47 分享**

```
台灣做永續，1–3 年月薪中位數 5.2 萬。新加坡同級職位，10.4–18.8 萬。

我原本以為只是物價差，直到把兩邊的工作內容放在一起看。

能力沒有突然變強。Scope 3 還是 Scope 3，ISO 14064 也沒到新加坡就變難。

差別是，很多台灣永續工作最後停在報告、查驗、揭露和合規，常被當成「不得不做的成本」。

到了新加坡，同一份碳數據可能接到融資、投資、採購、碳交易和風險決策。

一邊把報告交出去，一邊決定錢怎麼花。

市場不是按你有多努力定價，而是按你能影響多少錢定價。

所以真正該問的不是「要不要去新加坡」，而是：

你現在的專業，最後影響的是一份報告，還是一筆決策？
```

## 正典二（2026-07-12 發）：證照 544% 反差

**成效（48 小時）：1,250 views｜24 讚｜9 回覆｜0 轉發｜9 分享**

```
綠領薪資，比全市場只高 5.3%。

但有證照的人，
面試邀約卻成長了 544%。

我看到這兩個數字時，
愣了很久。

學碳盤查、考證照，
轉進大家口中的「未來產業」。

結果薪水多出來的，
只有 5.3%。

這幾年，我看過太多人為了這 5.3% 拼命。

再考一張證照。
把履歷塞滿 ISO 14064、GRI、TCFD。

我們都以為，
紙疊得夠高，薪水就會跟著上去。

但同一份報告裡，
還有另一個數字。

持有證照的人，
面試邀約成長了 544%。

這才是台灣綠領市場真正的現況：

市場現在獎勵的是「讓你進得去」，
還不是「讓你領得多」。

證照是開門用的。
不是加薪用的。

門打開之後，
真正決定你值多少的，
不是你有幾張紙。

而是你的永續能力，
接在什麼硬技能上。

碳核算接上財務。
合規接上供應鏈。
永續策略接上數據、採購或營運。

「永續」兩個字本身不值錢。

能解決什麼商業問題，
才值錢。

下一步真正該問的，
不是：

「我還缺哪一張證照？」

而是：

「我手上的牌，
可以接到市場最缺的哪個位置？」

轉發給那個還在猶豫，
要不要再考一張證照的朋友。
```

## 正典三（2026-07-14 發）：挖角那年 — **目前最弱**

**成效（API 實拉 2026-07-17）：281 views｜12 讚｜1 回覆｜0 轉發｜1 分享**

```
換工作那年，有人開了一個數字挖我。

我第一個念頭不是嫌低，而是：

「這麼多？我真的值嗎？」

我差點就答應。

後來才知道，那個數字比我當時該拿的低了一截。

我不是被騙。

是我根本不知道自己值多少。

沒有人教我，怎麼把「我做過的事」，翻成「市場願意付的價」。

這種虧，我不只吃過一次。

十年前，我在四大的辦公室，把一份永續報告改到第三版，卻不知道這條路到底有沒有前途。

想換工作時，我不知道自己的經驗接得到什麼職位。

決定念 MBA 前，我不會算獎學金、兩年薪水和轉職機會的整筆帳。

畢業找工作時，我甚至不知道該怎麼談薪水，換一個城市到底要花多少。

十年後回頭看，我真正缺的從來不是努力。

是每個岔路口，都沒有人先給我一張地圖。

所以我開始做一個網站。

把這十年用時間和錢換來的判斷，做成免費工具。

先幫你看清自己值多少、缺什麼、下一步往哪走。

真的需要有人陪你套到自己身上，再付費。

它從綠領職涯開始，因為這是我走過的路。

但它其實是做給每一個站在岔路口，不想再靠撞牆前進的人。

把這篇轉給那個正準備入行、接 offer、念書或換城市，卻不知道該怎麼選的人。
```

## 從三篇蒸餾的風格指紋（機器寫完後逐條對照）

1. **開場都是第一人稱的事實直述**（一個數字、一個場景），不是引言、不是問句、不是「在這個時代」
2. **冷靜拆解，不對抗**：正典裡沒有任何一句戰意宣言（「我不是靠 X 活下來的人」這種不是他）
3. **承認自己的不堪很自然**（「我真的值嗎」「我愣了很久」「這種虧我不只吃過一次」），但不賣慘
4. **對比句式「不是⋯而是⋯」用得節制**，出現在轉折或落點
5. **結尾遞給讀者一個問題或一個動作**，不喊口號
6. 全形標點，中英數字之間半形空格，斷行有節奏但不機械
