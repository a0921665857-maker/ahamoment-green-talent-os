# 故事型 Thread 劇本(Story-Thread Playbook)
*(zh-TW 為主 · Threads/X · 連結一律放第一則留言)*

> 核心:爆款故事 thread 不是「貼文」,是把讀者**丟進一個場景**,再讓他帶著改變走出來。Threads/X 的觸及靠 **停留時間**(會不會一直往下滑)+ **留言/轉發**(有沒有變成「他的」)。故事在兩者都贏:清單回答問題,故事讓人忘記正在被推銷、想知道「跟我一樣的人後來怎麼了」。

---

## 一、百萬流量故事 thread 的解剖

**1. 冷開場(第 1–2 行)— 最重要的一件事**
Threads 收合只露 ~2 行,所以第一行必須是**有時間/地點/身體感/數字的具體一刻**——不是論點、不是鉤子問句、不是「在這個時代…」。
三種有效開法:
- 時間+地點+身體:「凌晨兩點,我還在改一份沒人會看完的報告。」
- 一句對白(最強滑停,大腦會自動補「誰在說、說什麼」):「『你在四大查過財報吧?』面試官問。」
- 一個承載整篇的小物件:「我桌上那杯咖啡涼掉時,我才發現自己盯著同一頁 PPT 四十分鐘。」
反例(AI 味/雞湯):「轉職從來都不容易…」「你有沒有想過…」——這些是「感覺的摘要」,冷開場要是「感覺正在發生」。

**2. 帶入感鉤子(第 3–5 行)**
場景之後,馬上說出**內心狀態**(不是處境)——那句讀者也有過的沈默念頭。例:「第一個反應不是興奮,是慌:『我又沒有背景,誰要我?』」讀者共鳴的不是「四大、凌晨兩點」,是「我不夠格」。

**3. 五拍脊椎(背起來)**
① 場景 → ② 賭注(底下的恐懼/代價,那個「我也是」)→ ③ 假動作(你差點做的/大家都做的,製造張力:我也正要犯這錯)→ ④ 轉折(那句話/問題/領悟,最好是**別人的話**,外部真相更有力)→ ⑤ 落點(把領悟講白,再遞成讀者的鏡子)。
**轉折是彈頭。** 讀者要跟你**同時**愣住——那個被一起經歷的「幹,對耶」就是被轉發的瞬間。

**4. 張力 → 轉折 → 收束**
- 張力來自「假動作」(讀者正在做那件錯事)。
- 轉折要「意外、但事後覺得理所當然」。
- 收束**小而乾淨,絕不凱旋**:不是「然後我成功了!」,是安靜的身分句——「我不是沒有經驗的人,我是把審計腦袋帶進氣候的人。」(凱旋會打斷帶入感,讀者爬不上你的領獎台,只跟得進你的安靜領悟。)

**5. 至少埋 3 個「這就是我」觸發點**
說出口的羞恥念頭(「誰要我?」)/ 大家偷偷在做的浪費行為(報一堆證照、把主導講成參與)/ 身體反應(「我愣住。」身體反應是普世的,情緒是抽象的)/「你以為 X,其實 Y」只用**一次**在落點。

**6. 節奏與行長**
一行一個概念;空行就是節拍器(轉折前空一行=鼓聲);張力到頂時句子變短(領悟句最短);Threads 串接每則 1–4 短行,在懸崖處斷則(每次點開都是停留訊號)——自然斷點:假動作前、轉折前。

**7. 長度**
單則沉浸式 150–400 字,或 3–5 則串接(在張力點斷)。中段灌水是致命傷:每一行不是推進場景,就是加深感覺,否則砍掉。

**8. 點怎麼「賺」出來**
絕不說教(「所以我學到…」=帶入感死)。用發生在你身上的事**示範**,再用一句**鏡子**遞給讀者:「如果你也卡在『背景不對』——也許你只是還沒看懂,你的背景就是入場券。」

**9. 不破壞氛圍地接 CTA**
連結在第一則留言,所以**正文不賣**。正文收在情緒落點 + 一個**關係動作**(轉發給那個朋友/收藏/同意就轉)——這是關鍵:正文的「CTA」是人際手勢,不是產品。第一則留言才接產品,而且要**接住故事的同一條情緒線**(故事講「背景不對」→留言講「不確定你的背景能接到哪種綠領職位?」),像朋友遞出的下一句,不是廣告。

**10. 可偷的高手招式**
Humans of New York(一個陌生人、一個場景、一個安靜轉折,讓主角自己的話當轉折)、第一人稱職涯轉折的 LinkedIn 敘事(2am/失敗開場 → 重新定義落點,絕不炫耀)、對白開場+留白節奏的 thread 敘事派、「你以為…其實…」散文收尾(只用一次,重複=工整排比=AI 味)。

## 二、為什麼對「這個受眾」故事 > 清單/驚悚
受眾是「強但被誤讀」的亞太專業人——有能力,卻卡住、且為卡住默默羞愧。
1. **卡點是身分,不是資訊。** 他們不缺步驟,缺「相信自己的背景算數」的許可。清單做不了身分工作,故事可以。
2. **帶入感建立漏斗需要的信任**(終點是真人通話)。被故事「看見」過,第一則留言才像朋友伸手,不像廣告。
3. **亞太的內斂吃安靜誠實,不吃成功學**;驚悚換到的是錯的流量(酸民不會貼 CV)。
4. **單兵創辦人的護城河是「你本人」**——故事複利你的真實轉職經歷,清單是商品、無法複製。
**陷阱:** 偽謙虛炫耀(落點落在「我進了 Sylvera ✨」)、沒有「我也是」鉤子(變日記)、結尾說教清單、假脆弱/硬湊戲劇(這受眾+你都聞得出 AI 味)、工整排比、沒有賭注(沒有代價就沒人往下滑)。

## 三、可重複的故事 thread 模板
```
【拍1 場景 1–2行】第一行=滑停。時間+地點+身體,或一句對白,或一個物件。不要論點/問句/「在這個時代」。
【拍2 內心狀態 1–2行】底下那句沈默念頭(說出羞恥的那個)。讀者共鳴的是感覺,不是處境。
【拍3 假動作 2–3行】你差點做的/大家都做的。讀者正在做這件事=張力。
   ⟶(要分則就在這裡斷)
【拍4 轉折 2–4行】前面空一行。那句話/問題/領悟,最好是別人的話。把反應獨立一行:「我愣住。」再給反轉(意外→理所當然)。
【拍5 落點 2行】安靜的身分句,不是勝利。「我不是 X 的人,我是 Y 的人。」
【拍6 鏡子 1–2行】「如果你也卡在___,也許你只是還沒看懂,___。」遞出去,不說教。
【拍7 關係動作 斜體1行】轉發給那個[具體的人]/收藏/同意就轉。是人際手勢,不是產品。
──────────
【第一則留言 產品橋接】接住上面同一個情緒詞 + 免費測評一句話 + 👉 連結。像故事的下一句。
```
兩條鐵律:「不是…是…」結構**只在落點用一次**;每行不推進場景就加深感覺,否則刪。

## 四、兩篇可直接發的故事 thread(取自你真實素材,連結放第一則留言)

### 故事 A — 你自己的轉職(凌晨改 ESG 報告 → 面試官那一問 → 碳市場)｜建議 21:00–22:00
```
凌晨兩點,辦公室只剩我桌上那盞燈。

我在改一份 ESG 報告的第 47 頁。
改完我心裡很清楚:這份東西,客戶會收下、會歸檔,
然後沒有人會讀到第 47 頁。

那一刻我沒有崩潰,也沒有熱血。
只有一個很安靜的念頭浮上來:
我到底,在幫誰省什麼。

我想做真正會改變什麼的氣候工作。
但下一秒,慌就追上來了——
「我又沒有環境背景。私立大學、人文科系、
連 GMAT 都考得普普。誰會要我?」

於是我做了大多數人會做的事。
我開始查證照。一張、兩張、三張。
甚至認真想:要不要乾脆再去念一個環境的學位?
用一張新的紙,蓋掉我覺得自己不夠格的地方。

直到一場面試。
對方看著我的履歷,問了一句我完全沒準備的話:

「你在四大查過財報吧?
那你看得懂——一家公司的碳數據,是真是假嗎?」

我愣住。

我準備了一整套「我為什麼想轉氣候」的說詞,
卻沒想過,我以為的弱點,
正是這個產業現在最缺的東西:
一個會懷疑數字、會追到底的腦袋。

我不是一個沒有背景的人。
我是把審計的眼睛,帶進氣候的人。

後來我真的轉進了碳市場。
靠的不是那幾張我差點去考的證照,
是有人幫我把「我有的」,接到「市場缺的」上面。

如果你也卡在「我背景不對」——
也許你不是不夠格。
是還沒有人,幫你把手上的牌,翻成市場讀得懂的那一面。

(轉發給那個一直說想轉氣候、卻總覺得自己不夠格的朋友。)
```

### 故事 B — 強但被誤讀(那個輸給「比較好懂」的人的朋友)｜建議 21:00–22:00
```
一個朋友傳訊息給我:「我又沒上。」

我認識他。他真的很強——
做過三個碳盤查專案,英文好,人也踏實。
比我當年強多了。

我跟他要了那份履歷。
看完我大概知道,他輸在哪了。

不是經驗。
是我讀到第三遍,才搞懂他到底會解決什麼問題。

他把「主導一個跨部門減碳專案」,
寫成「協助永續相關業務之推動」。
他把幫公司省下的真金白銀,
寫成「負責碳管理事務」。

每一個字都是真的。
每一個字都在幫他扣分。

我問他:你為什麼這樣寫?
他說:「我怕寫得太滿,被說在邀功。」

那一句,我聽過太多次了。

我們這種人,從小被教要謙虛、要把功勞讓出去。
所以履歷上,我們本能地把「主導」縮成「參與」,
把「策略」縮成「執行」,把自己縮到最小。

然後我們納悶,為什麼比較不厲害的人,先拿到了 offer。

招募的人沒有時間,幫你把散落的價值拼回去。
他三秒讀不懂你,你再強,對他來說都不存在。

我幫那個朋友改了履歷。
沒有加任何一個字的假話。
只是把他本來就做到的事,用市場讀得懂的方式,說出來。

兩週後,他面上了。

他不是變強了。
他只是,終於被讀懂了。

如果你也總覺得「我明明很強,怎麼都沒人看見」——
也許你缺的不是能力。
是有人幫你,把你翻成市場讀得懂的版本。

(收藏起來。下次投履歷前,先問自己:對方三秒,讀得懂我嗎?)
```
